# Sijiwa
Aplikasi kesehatan mental yang membantu pengguna memahami dan memantau keseimbangan emosi, terutama bagi individu dengan gangguan bipolar atau yang ingin mengenali gejalanya sejak dini. Output utamanya berupa data, grafik, laporan, dan rekomendasi kesehatan mental untuk mengenali pola emosi serta memudahkan konsultasi dengan profesional.
